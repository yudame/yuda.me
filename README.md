# Yudame

🌐 **Live at**: [yudame.org](https://yudame.org)

Business website for Yudame - AI-forward product development.

## Quick Start

```bash
# Install dependencies
npm install

# Development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- HTML5 + Tailwind CSS
- GitHub Pages
- GitHub Actions (CI/CD)
